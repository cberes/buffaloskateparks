## Location

8787 Erie Rd, Angola, NY 14006

## Description

Modular skatepark with a decent asphalt surface. The ramps are mellow, which is good for begineers. However, most of the features need some maintenance. 

## Features

- 3' and 4' quarter pipes
- funboxes
- box
- square flat rails
- 3' bank
